
# 📰 Nabd Masr (نبض مصر) - Egypt News Feed

A dynamic and responsive frontend application designed to deliver the latest and most important news in Egypt. Built with modern web technologies, it ensures a seamless user experience with clean architecture and Google's Material Design principles.

## ✨ Key Features

* **Real-time News Feed:** Fetches the latest articles dynamically via REST API.
* **Advanced Search:** Quick and accurate search functionality to find specific news topics.
* **Category Filtering:** Filter news by different categories to customize the feed.
* **Seamless Pagination:** Smooth navigation through hundreds of articles without compromising performance.
* **Material Design UI:** A sleek, user-friendly, and fully responsive interface powered by Material UI (MUI).

## 🛠️ Tech Stack

* **Framework:** React (via Vite for blazing-fast performance)
* **Language:** TypeScript
* **UI Library:** Material UI (MUI)
* **Styling/Design System:** Google Material Design
* **Data Fetching:** REST API Integration

## 🚀 Getting Started

To run this project locally:

### 1. Clone the repository
```bash
git clone git@github.com:N3ssar/nabd-masr.git
```

### 2. Navigate to the project directory
```bash
cd nabd-masr
```

### 3. Install dependencies
```bash
npm install
```

### 4. Start the development server
```bash
npm run dev
```
